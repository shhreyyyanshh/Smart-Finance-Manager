
# Smart Personal Finance Manager

## Overview
A full-stack finance management system built using Java, Spring Boot, and MySQL with a Swing-based desktop interface.

## Features
- Add income and expenses
- Store data in MySQL database
- View transactions in table format
- REST API-based architecture

## Tech Stack
- Java (OOP)
- Spring Boot
- MySQL
- Java Swing

## Architecture
Swing UI → REST API → Spring Boot → MySQL

## How to Run

### Backend:
Run Spring Boot:
mvnw.cmd spring-boot:run

### Frontend:
Run Swing App:
javac FinanceSwingApp.java
java FinanceSwingApp
